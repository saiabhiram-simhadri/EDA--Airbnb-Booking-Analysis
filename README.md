# EDA---Airbnb-Booking-Analysis
An exploratory data analysis conducted on Airbnb booking data

# Input Files
- Airbnb_analysis.csv - The listing details of around 49k providing information of various aspects of booking.

# Introduction
In this analysis we are targeting to understand all the aspects of how the bookings are impacting the hosts and what actually matters if we can isolate the key metrics critical for the pricing.
